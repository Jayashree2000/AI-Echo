# 🤖 AI Echo – Intelligent Text Analysis & Sentiment Prediction

AI Echo is a Natural Language Processing (NLP) project designed to analyze text data and predict sentiment using Machine Learning techniques. The system processes user input and classifies it into **Positive, Negative, or Neutral sentiment**, helping understand opinions and feedback from textual data.

This project demonstrates the practical implementation of **NLP, text preprocessing, feature extraction, and machine learning models** for sentiment analysis.

---

## 🚀 Features

* 🧠 Sentiment classification (Positive / Negative / Neutral)
* 🔎 Text preprocessing and cleaning
* 📊 Feature extraction using **TF-IDF**
* 🤖 Machine Learning based prediction
* ⚡ Fast and simple text analysis
* 📈 Model evaluation with performance metrics
